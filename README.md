## ✨ Características

- 📊 **Dashboard** con KPIs del portafolio, ingresos mensuales y estado de ocupación
- 🏘️ **Propiedades** con estado (arrendada / mora / disponible), datos del inquilino y vencimiento
- 💰 **Pagos** con resumen por estado (cobrado / pendiente / mora) y marcado en un click
- 🔧 **Mantenimiento** con prioridad (alta/media/baja), técnico asignado y resolución
- 📈 **Reportes** con ingresos por propiedad, depósitos, proyección anual y tasa de ocupación
- 🔍 **Búsqueda y filtros** por nombre, inquilino y estado
- 🌗 **Diseño dark futurista** con paleta cyan-violeta
- 📥 **Exportación CSV** de propiedades, pagos y mantenimientos

## 🛠️ Stack tecnológico

- **React 18** — hooks, `useMemo`, `useCallback`
- **Vite 5** — bundler con HMR
- **CSS-in-JS** — estilos inline con sistema de tema
- **Gráficas custom** — sin librerías pesadas, sólo CSS
- **GitHub Actions + GitHub Pages** — CI/CD

## 💻 Instalación local

```bash
git clone https://github.com/Juanmaya25/renttrack-rentals.git
cd renttrack-rentals
npm install
npm run dev
```

Abre [http://localhost:5173/renttrack-rentals/](http://localhost:5173/renttrack-rentals/).
